# MPTech Network Tools v1.0.0

Portable Windows network diagnostics toolkit by MPTech Tools.

## Fix included

- Fixed Spanish and Portuguese UTF-8 text encoding.
- Removed broken mojibake characters from the UI.
- Cleaned accidental backup files from the repository.

## Included modules

- Network summary
- Diagnostic
- Ping
- TCP port test
- Local listening ports
- Network Scan
- Traceroute
- Report copy/export
- English, Spanish and Portuguese UI
